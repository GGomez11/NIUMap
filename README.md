# NiuMap

Personal project that visualized shortest paths on the NIU compass. Program also displays agents traversing the campus. 

## Package Installation
```julia
using Pkg; Pkg.add("OpenStreetMapXPlot", "OpenStreetMapX", "Plots")
```
